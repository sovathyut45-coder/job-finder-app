# 💼 Job Finder App

A professional Flutter Job Finder application built with GetX and Repository Pattern.

## 📱 Features

### Job Search
- Search Jobs
- Debounce Search
- Popular Job Categories
- Search Suggestions

### Job Browsing
- Popular Jobs
- Job Details
- Company Information
- Apply Job

### User Features
- Save Jobs
- Favorites Screen
- Pull To Refresh
- Pagination (Infinite Scroll)

### UI/UX
- Responsive Design
- Clean Architecture
- Loading States
- Error Handling
